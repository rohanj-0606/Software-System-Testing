1. account.jar is the executable files.

2. You need to have Java Runtime Environment (JRE) 1.6 or above to run the executable jar files.

3. To run the account.jar file on Mac OS, type the following on terminal and perform required operations by giving the displayed input number to execute the test cases.

	java -jar account.jar

4. To run the account.jar file on Windows OS, type the following on command prompt and perform required operations by giving the displayed input number to execute the test cases.
	
	java -jar account.jar
